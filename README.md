💰 Finance & Expense Tracker Using SQL
📌 Project Overview

The Finance & Expense Tracker project is a SQL-based data analytics project designed to monitor personal financial activities and generate meaningful insights into income, expenses, savings, budgeting, and cash flow. It helps analyze spending habits, compare budgets with actual expenses, and identify opportunities to improve financial management.

This project demonstrates how SQL can be used to solve real-world financial analysis problems using relational databases and business reporting techniques.

Developed as part of my SQL Developer Internship at ELiteTech Intern, this project strengthened my SQL querying, database management, and financial analytics skills.

🎯 Project Objectives
Design a relational finance database.
Track monthly income and expenses.
Compare budget vs actual spending.
Analyze savings.
Monitor category-wise spending.
Generate cash flow reports.

Practice advanced SQL concepts through financial analytics.
